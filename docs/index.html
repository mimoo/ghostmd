Your notes are text files. That's it.

native macOS · free · open source

The opinions

  • Notes are ~/Documents/ghostmd/*.md. Not a database. Not synced to our servers. Files on your disk, forever yours.
  • Markdown is already beautiful. We don't render it. We don't preview it. You read it raw, the way it was meant to be.
  • No images. If you need images, you need a different tool. This is for words.
  • No config files. No themes. No plugins. We made the choices so you can focus on making the words.
  • Diary entries get a path: diary/2026/march/03/. You don't pick it. The date does.
  • AI helps you think, not format. Titles, search, suggestions — never decoration.
  • Native macOS, GPU-accelerated via Metal. Opens before your finger leaves the key. Quits without asking.

# Building ghostmd

*A note-taking app that gets out of the way.*

## Why plain markdown?

Every note app eventually dies. When it does, your notes
should survive. Plain .md files will outlive every app,
every company, every format war.

- No database migrations
- No export button needed
- grep works. git works. Everything works.
- **Your files. Your disk. Your rules.**

See [the repo](https://github.com/...) for more.

Raw markdown. No preview pane. That's the point.

What you'll never get

No config No Electron No cloud No image support No preview pane No plugins No themes No mobile app No export menu No WYSIWYG No telemetry No account

Every feature we didn't build is time you spend writing.

What you get

  • Workspaces & split panes
  • Fuzzy file finder
  • Full-text search across all notes
  • Agentic AI search
  • Branching undo tree — not just ctrl-z
  • Auto-save on every keystroke
  • AI-generated titles
  • Session persistence — reopen exactly where you left off
  • Your files, forever readable, even if we disappear tomorrow
